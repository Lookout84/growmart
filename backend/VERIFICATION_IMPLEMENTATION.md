## 🚀 Як використовувати

### Верифікація через адмін панель:

1. **Откройте адмін панель:**
   - URL: `http://localhost:8000/admin/`
   - Username: `admin`
   - Password: `admin123`

2. **Перейти до розділу Користувачі**

3. **Масова верифікація:**
   - Виберіть користувачів (позначте чекбокси)
   - У меню "Дії" виберіть "✓ Верифікувати вибраних користувачів"
   - Натисніть кнопку "Перейти"

4. **Індивідуальна верифікація:**
   - Натисніть на користувача
   - Позначте чекбокс "Верифіковано"
   - Натисніть "Зберегти"

### Програмна верифікація (для тестування):

```bash
# Верифікувати користувача
docker compose exec backend python manage.py shell -c "
from django.contrib.auth import get_user_model
User = get_user_model()
user = User.objects.get(username='Lookout')
user.is_verified = True
user.save()
"

# Скасувати верифікацію
docker compose exec backend python manage.py shell -c "
from django.contrib.auth import get_user_model
User = get_user_model()
user = User.objects.get(username='Lookout')
user.is_verified = False
user.save()
"
```
