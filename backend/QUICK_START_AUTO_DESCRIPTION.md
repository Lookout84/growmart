# 🤖 Швидкий старт: Автогенерація описів товарів

## Можливості

✅ Автоматична генерація повного та короткого опису товару  
✅ Підтримка AI генерації (OpenAI GPT) та шаблонної генерації  
✅ Автоматична генерація артикулу (SKU)  
✅ Працює через адмін-панель та API  

---

## Як використовувати

### Варіант 1: Через Адмін-панель (найпростіше)

1. Відкрийте http://localhost:8000/admin/products/product/add/
2. Заповніть **тільки**:
   - Назву товару
   - Категорію
   - Ціну
   - Залишок
3. **Не заповнюйте** поля "Опис", "Короткий опис", "Артикул", "URL"
4. Натисніть "Зберегти"
5. ✨ Опис згенерується автоматично!

### Варіант 2: Масова генерація для існуючих товарів

1. Перейдіть до списку товарів: http://localhost:8000/admin/products/product/
2. Виберіть товари (checkboxes)
3. В меню "Дії" оберіть "📝 Згенерувати описи (Шаблон)"
4. Натисніть "Виконати"
5. ✨ Описи згенеруються для всіх вибраних товарів!

### Варіант 3: Через API

```bash
curl -X POST http://localhost:8000/api/products/ \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "name": "Яблуня Голден Делішес",
    "category": 1,
    "price": 150.00,
    "stock": 50
  }'
```

## Налаштування AI генерації (опціонально)

> ⚠️ За замовчуванням використовується **шаблонна генерація** (працює без налаштувань)

Для AI генерації потрібен API ключ OpenAI:

1. Отримайте ключ на https://platform.openai.com/
2. Додайте в `docker-compose.yml`:

```yaml
backend:
  environment:
    - OPENAI_API_KEY=sk-your-api-key-here
```

3. Перезапустіть контейнери:

```bash
docker-compose restart backend
```

## Що генерується автоматично?

✅ **Опис товару** (description) - детальний опис 150-250 слів  
✅ **Короткий опис** (short_description) - 1-2 речення  
✅ **Артикул** (SKU) - формат `SKU-XXXXXXXX`  
✅ **URL** (slug) - автоматично з назви товару
