# 🎉 Нові функції GrowMart

## Що додано (29.03.2026)

### 1️⃣ Автоматична генерація артикулів (SKU)
При створенні товару без артикулу, система автоматично генерує унікальний код формату `SKU-XXXXXXXX`

**Приклад:** `SKU-A5B9C2D1`

### 2️⃣ Автоматична генерація описів товарів
При створенні товару система може автоматично згенерувати:
- Повний опис (150-250 слів)
- Короткий опис (1-2 речення)

**Два режими:**
- 🤖 **AI генерація** - через OpenAI GPT (опціонально, потрібен API ключ)
- 📝 **Шаблонна генерація** - працює одразу без налаштувань

### 3️⃣ Автоматичний пошук та додавання зображень (НОВЕ!)
Система автоматично знаходить та додає професійні фотографії товарів:

**Три джерела:**
- 🖼️ **Unsplash** - безкоштовний фотосток (мільйони фото рослин)
- 🖼️ **Pexels** - альтернативний безкоштовний фотосток
- 🎨 **DALL-E** - AI генерація унікальних зображень (OpenAI)

**Можливості:**
- Пошук 1-10 зображень автоматично
- Оптимізація розміру (до 1920x1920px, JPEG, 85% якість)
- Переклад українських назв на англійські для кращого пошуку
- Масове додавання зображень через адмін-панель

## Швидкий старт

### Через адмін-панель

1. Відкрити http://localhost:8000/admin/products/product/add/
2. Заповнити **тільки**:
   - Назву товару
   - Категорію
   - Ціну  
   - Залишок
3. **Не заповнювати** опис та артикул
4. Зберегти
5. ✨ **Опис та артикул згенеруються автоматично!**

### Через API

```bash
curl -X POST http://localhost:8000/api/products/ \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "name": "Яблуня Голден Делішес",
    "category": 1,
    "price": 150.00,
    "stock": 50,
    "auto_add_images": true,
    "image_count": 3
  }'

# Відповідь:
{
  "id": 1,
  "name": "Яблуня Голден Делішес",
  "sku": "SKU-A5B9C2D1",           # ✅ Згенеровано автоматично
  "slug": "yablunia-golden-delishes",
  "description": "..." ,           # ✅ Згенеровано автоматично
  "short_description": "...",      # ✅ Згенеровано автоматично
  "images": [...]                  # ✅ Додано автоматично (якщо auto_add_images: true)
  "price": "150.00"
}
```

### Додавання зображень до існуючих товарів

**Через адмін-панель:**
1. Відкрити http://localhost:8000/admin/products/product/
2. Вибрати товари (checkboxes)
3. Дія: **"🖼️ Додати зображення (Unsplash/Pexels)"** або **"🎨 Згенерувати зображення (DALL-E AI)"**
4. Виконати
5. ✨ Зображення додадуться автоматично!

## Налаштування AI генерації (опціонально)

За замовчуванням використовується **шаблонна генерація** (працює одразу).

Для AI генерації через OpenAI GPT:

1. Отримати API ключ на https://platform.openai.com/
2. Додати в `docker-compose.yml`:

```yaml
backend:
  environment:
    - OPENAI_API_KEY=sk-your-api-key-here
```

3. Перезапустити:
```bash
docker-compose restart backend
```

## FAQ

**Q: Працює без OpenAI API ключа?**  
✅ Так! За замовчуванням використовується шаблонна генерація

**Q: Можна редагувати згенерований опис?**  
✅ Так, просто відредагуйте в адмін-панелі

**Q: Що якщо я вказав опис вручну?**  
✅ Автогенерація не спрацює, використається ваш опис

**Q: Скільки коштує OpenAI API?**  
💰 Приблизно $0.001-0.005 за один опис

**Q: Можна вимкнути автогенерацію?**  
✅ Так, передайте `"auto_generate_description": false` в API
