# Зелений куточок - Інтернет-магазин саджанців 🌱

Інтернет-магазин для продажу саджанців плодових дерев, ягідних культур, винограду та декоративних рослин. Проект перероблений за дизайном сайту [litynsad.com.ua](https://litynsad.com.ua).

## 🌿 Основні можливості

### Каталог товарів
- **Плодові дерева** - яблуні, груші, сливи, черешні
- **Ягідні чагарники** - малина, смородина, аґрус  
- **Виноград** - столові та технічні сорти
- **Полуниця** - ремонтантні та класичні сорти
- **Троянди** - паркові, чайно-гібридні, плетисті
- **Декоративні рослини** - туї, ялівці, бузок
- **Екзотичні рослини** - інжир, ківі, гранат
- **Колоновидні саджанці**

### Функціонал
- 🎯 Банери з акціями та сезонними пропозиціями
- 🌟 Новинки та бестселлери
- 📝 Блог з порадами по вирощуванню
- 🧮 Калькулятор кількості рослин для ділянки
- ❤️ Список бажань (Мій сад)
- 📦 Кошик та оформлення замовлення
- 👤 Особистий кабінет
- 💳 Оплата онлайн
- 🚚 Інформація про доставку

### Характеристики товарів
- Сорт (ранній, середній, пізній)
- Вік саджанця
- Маса плоду
- Термін дозрівання
- Сезон посадки
- Тип упаковки (відкритий корінь, в горщику)
- Код товару
- Статус наявності

## 📋 Встановлення та запуск

### Через Docker (рекомендовано)

1. **Клонуйте репозиторій:**
```bash
git clone <repository-url>
cd growmart
```

2. **Запустіть проект:**
```bash
docker compose up --build
```

3. **Доступ до сервісів:**
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- Admin Panel: http://localhost:8000/admin

### Локальна розробка

#### Backend

1. **Створіть віртуальне середовище:**
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac
```

2. **Встановіть залежності:**
```bash
pip install -r requirements.txt
```

3. **Налаштуйте змінні оточення:**
Створіть файл `.env` в папці backend:
```
DEBUG=True
SECRET_KEY=your-secret-key
DB_NAME=growmart
DB_USER=postgres
DB_PASSWORD=postgres
DB_HOST=localhost
DB_PORT=5432
REDIS_URL=redis://localhost:6379/1
CELERY_BROKER_URL=redis://localhost:6379/0
```

4. **Виконайте міграції:**
```bash
python manage.py migrate
python manage.py createsuperuser
python manage.py collectstatic
```

5. **Запустіть сервер:**
```bash
python manage.py runserver
```

#### Frontend

1. **Встановіть залежності:**
```bash
cd frontend
npm install
```

2. **Налаштуйте API URL:**
Створіть файл `.env` в папці frontend:
```
REACT_APP_API_URL=http://localhost:8000
```

3. **Запустіть додаток:**
```bash
npm start
```

## 🗄 Структура бази даних

### Основні моделі:

**Product (Товар)**
- Назва, опис, ціна
- Категорія
- Характеристики саджанців (сорт, вік, вага плоду, термін дозрівання)
- Статуси (новинка, бестселер, популярний, акція)
- Інформація про доставку

**Category (Категорія)**
- Ієрархічна структура
- Зображення

**Banner (Банер)**
- Акційні пропозиції
- Сезонні кампанії

**BlogPost (Пост блогу)**
- Корисні статті
- Поради по вирощуванню
- Категорії блогу

**Wishlist (Список бажань)**
- Збереження улюблених товарів
