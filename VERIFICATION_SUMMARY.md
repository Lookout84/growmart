## 🚀 Как использовать

### Способ 1: Индивидуальная верификация
1. Откройте админ-панель: `http://localhost:8000/admin/`
2. Перейдите в "Пользователи"
3. Нажмите на пользователя
4. Отметьте чекбокс "Верифицировано"
5. Нажмите "Сохранить"

### Способ 2: Массовая верификация
1. На странице списка пользователей
2. Выберите пользователей (чекбоксы слева)
3. В меню "Действия" выберите "✓ Верифицировать выбранных пользователей"
4. Нажмите "Перейти"

### Способ 3: Программно (для разработчиков)
```bash
docker compose exec backend python manage.py shell -c "
from django.contrib.auth import get_user_model
User = get_user_model()
user = User.objects.get(username='Lookout')
user.is_verified = True
user.save()
"
```
