# 🚀 QUICK START - Верифікація користувачів

## 1️⃣ Вхід в адмін панель

```
URL: http://localhost:8000/admin/
Username: admin
Password: admin123
```

## 2️⃣ Перейти до розділу Користувачі

1. Натисніть "Користувачі" в лівому меню
2. Побачите список всіх користувачів

## 3️⃣ Варіант A: Індивідуальна верифікація

1. Натисніть на прізвище користувача
2. Знайдіть розділ "Верифікація та контакти"
3. Позначте чекбокс "Верифіковано"
4. Натисніть кнопку "Зберегти"

## 3️⃣ Варіант B: Масова верифікація

1. На сторінці списку користувачів
2. Виберіть користувачів, позначивши чекбокси на лівій стороні
3. У меню "Дії" вибрати:
   - `✓ Верифікувати вибраних користувачів` - для верифікації
   - `✗ Скасувати верифікацію вибраних користувачів` - для скасування
4. Натисніть кнопку "Перейти"

## 4️⃣ Фільтрація користувачів

На сторінці списку користувачів можна вибрати фільтри:
- **is_verified: Yes** - показати тільки верифікованих
- **is_verified: No** - показати тільки не верифікованих

## 5️⃣ Статус верифікації

У списку користувачів видно статус:
- ✅ **Верифіковано** - користувач активний
- ❌ **Не верифіковано** - очікує верифікації

## 💡 Типовий робочий процес

```
1. Користувач реєструється через фронтенд
   ↓
2. Адміністратор перевіряє дані користувача в адмін панелі
   ↓
3. Адміністратор верифікує користувача (позначає чекбокс)
   ↓
4. Користувач отримує емейл про успішну верифікацію
   ↓
5. Користувач отримує повний доступ до платформи
```

## 📧 Емейл при верифікації

Коли ви верифікуєте користувача, на його email автоматично відправляється письмо з:
- Повідомленням про успішну верифікацію
- Списком доступних функцій
- Посиланням на сайт

## ⚙️ Програмна верифікація (для розробників)

Якщо потрібна програмна верифікація:

```bash
# Верифікувати користувача
docker compose exec backend python manage.py shell -c "
from django.contrib.auth import get_user_model
User = get_user_model()
user = User.objects.get(username='Lookout')
user.is_verified = True
user.save()
print('✅ Користувач верифіков!')
"

# Скасувати верифікацію
docker compose exec backend python manage.py shell -c "
from django.contrib.auth import get_user_model
User = get_user_model()
user = User.objects.get(username='Lookout')
user.is_verified = False
user.save()
print('✅ Верифікація скасована!')
"
```
